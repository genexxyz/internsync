# InternSync

**InternSync** is a web-based On-the-Job Training (OJT) monitoring system designed for colleges to streamline the management of internship programs. It features application tracking, attendance monitoring, notification systems, and auto-generated reports to enhance the OJT experience for students and facilitators.

---

## Features

- **Application Tracking**: Monitor student applications to companies.  
- **Document Management**: Handle OJT documents with ease.  
- **Progress Monitoring**: Track attendance and progress during OJT. 
- **Notifications**: Stay updated with notifications for important events.  
- **Auto-Generated Reports**: Generate detailed reports for OJT records.  

---

## Usage

1. Access the application in your browser.  
2. Log in as a facilitator or student to manage and monitor OJT processes.  
3. Use the admin panel for advanced management features.  

---

## Technologies Used

- **Framework**: Laravel 11, Livewire  
- **Frontend**: Blade Templates, TailwindCSS  
- **Database**: MySQL  
- **Other Tools**: XAMPP for local development, Git for version control
